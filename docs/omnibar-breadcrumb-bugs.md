# Omnibar / PathBreadcrumbBar — 回归清单

实现已对齐 Files 核心行为，请按下列项**统一手工测试**。

## 导航

- [ ] 左键：点击非最后路径段 → 进入该目录，面包屑与列表同步
- [ ] 左键：点击最后一段 → 不导航
- [ ] 中键：非最后段 → **新标签**打开该路径
- [ ] 根「首页」图标 → 进入 Home（无重复 Home 段）
- [ ] `…` 菜单项 → 进入对应前缀目录
- [ ] 段内 `›` 下拉 → 子文件夹列表；**不含当前目录项**；尊重「显示隐藏文件」
- [ ] 根 `›` 下拉 → 快速访问 + 驱动器（每次打开为最新 pinned）

## 折叠与布局

- [ ] 窗口变窄 → 隐藏**左侧**前缀，保留尾部（非中间截断）
- [ ] 拉宽窗口 → 省略号消失
- [ ] 无横向滚动条

## Omnibar 模式

- [ ] 失焦 → 面包屑 + Path 模式
- [ ] 点击面包屑空白 → 路径编辑
- [ ] 点击段/根/省略号 → 不误进编辑
- [ ] `Ctrl+L` → 聚焦路径输入
- [ ] `Escape`（编辑中）→ 回到面包屑
- [ ] Enter 提交路径；Settings/Recycle 伪路径段可点

## 拖放

- [ ] 将文件拖到**非最后**目录段 → 悬停时预览进入该目录（与 Files 类似）
- [ ] 放到目录段上 → 复制（Ctrl）或移动到该文件夹
- [ ] 最后一段（当前目录）→ 可 drop，不触发悬停导航

## 稳定性

- [ ] 工具栏 **后退/前进/向上** → 无 panic（`location_changed` 已 defer，避免嵌套 `MainPage::update`）
- [ ] 侧栏 / 面包屑快速切换 → 同上
- [ ] 无其它 panic

## 已知差异（可后续迭代）

- Chevron 展开时动态旋转（当前为固定向下 `›`）
- 面包屑宽度为窗口估算，非逐段实测
- 根/子项 Shell 缩略图
- 拖放悬停无 350ms 防抖（立即预览导航）
