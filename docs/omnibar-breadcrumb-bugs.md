# Omnibar / PathBreadcrumbBar — 回归清单

## 导航

- [ ] 左键：非最后段 → 进入目录
- [ ] 左键：最后段 → 不导航
- [ ] 中键：非最后段 → 新标签
- [ ] 根首页图标 → Home
- [ ] `…` / 段 `›` 下拉；不含当前目录；尊重隐藏项
- [ ] 无权限目录 → 菜单显示「访问被拒绝」（非「没有子文件夹」）

## 路径栏模式

- [ ] 点击面包屑**空白** → 路径输入框、光标、可编辑
- [ ] 点击段/根/`…` → 不进编辑
- [ ] Enter 提交路径；Esc / 点标题栏或列表空白 → 回面包屑
- [ ] **Ctrl+L** → 聚焦右侧**搜索框**（非路径栏）

## 拖放

- [ ] 拖到非最后段悬停 **~1.3s** 后预览进入
- [ ] 最后段可 drop，无悬停导航

## 文件列表

- [ ] 打开目录默认无选中高亮
- [ ] 点列表空白 → 取消选中
- [ ] 空白处右键 → 粘贴 / 新建文件夹 / 新建文件 / 刷新

## 稳定性

- [ ] 工具栏后退/前进/向上无 panic
- [ ] 快速切换路径无 panic
